# MyDesk v1.3.0

MyDesk v1.3.0 is a minor release focused on workspace-level resource reuse, a stronger workspace todo board, and clearer release documentation.

## Product Direction

MyDesk continues to treat the local file system as the source of truth. This release improves the layer above it: resources can be understood by where they are used, workspace canvases can reference more of the shared library, and project tasks can live closer to the visual workflow they support.

## New and Improved

- Added workspace todo groups so tasks can be organized into project sections instead of a single flat list.
- Added task details, pinned tasks, creation dates, optional DDL dates, and linked resources.
- Added a task detail editor for title, notes, dates, and resource linkage.
- Improved task ordering with pinned-first sorting, stable sort indexes, drag reordering, and focused regression tests.
- Added Global Library workspace-usage display, including direct workspace resources and resources already placed on canvases.
- Added a workspace filter in Global Library so users can quickly see which resources are used by a selected workspace.
- Improved global resource deduplication so global pins and workspace-scoped pins can coexist without accidental merging.
- Updated canvas resource insertion so workspace canvases can add global resources, current workspace resources, and resources from other workspaces through grouped menus.
- Improved workspace cleanup so deleting a workspace also removes its todo groups.
- Refined navigation titles, layout spacing, and lower status-bar density.
- Updated README with versioned package names, bilingual release details, and badge-based language switching.

## Fixes

- Fixed resource import deduplication treating identical paths from different workspace scopes as the same record.
- Fixed Global Library hiding workspace-scoped resources that should still appear as reusable library records.
- Fixed workspace todo board behavior around completed-task ordering and group-aware sorting.
- Fixed stale todo group records remaining after workspace deletion.
