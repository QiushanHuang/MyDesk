# MyDesk v1.1.0

MyDesk v1.1.0 improves the workspace canvas so it is easier to document, shape, and preserve visual workflows.

## Current Features

- Native macOS workbench for organizing folders, files, snippets, prompts, commands, notes, and workflow maps.
- Home dashboard for reopening active workspaces and pinned resources.
- Workspace management with create, rename, delete, pin, sort, and quick reopen actions.
- Global Library for reusable file and folder sources.
- Pinned Folders and Pinned Files for fast access to important resources.
- Finder integration for opening folders, revealing files, copying full paths, and creating aliases after confirmation.
- Snippet Library for reusable prompts, commands, notes, and text blocks.
- Workspace Canvas with resource cards, note cards, organization frames, zoom controls, box selection, auto-arrange, alignment, arrow links, and animated flow lines.
- Inline canvas editing for note card titles, note descriptions, organization frame names, and frame notes.
- Draggable link bend handles for manually shaping workflow connections.
- JSON import/export for backup and portability.
- MIT license and reproducible macOS packaging scripts.

## New in v1.1.0

- Added draggable link bend handles on canvas connections.
- Saved custom link bend control points in SwiftData.
- Preserved custom link bend control points in JSON export/import.
- Added inline editing for note card titles and note descriptions.
- Added inline editing for organization frame names and frame notes.
- Added coordinate-conversion and custom edge-control tests in `MyDeskCoreTests`.
- Updated packaging scripts so build numbers are derived from the app version.

## Distribution

- macOS 14 or newer.
- Ad-hoc signed. This release is not notarized because no Developer ID certificate is configured for this repository yet.
